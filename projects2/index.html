2. Добавьте с свойства для div.task-2 такие, чтобы получить прямоугольник высотой 100рх, шириной 250рх, цвет фона brown, border 2x шириной solid, цвет coral.

3. Допишите еще один класс round блоку div.task-1 приведенному ниже. Добавьте в CSS класс round, со свойствами: border-radius: 50%;

4. Создайте div.task-4 и присвойте ему свойства в CSS: background: rgb(255, 127, 80). Изучите полученный цвет. Ширина блока 100рх, высота 75рх.

5. Блоку div.task-5 приведенному ниже, добавьте класс task-4. Блок должен окрасится в цвет сога. Для класса task-5 добавьте стиль гдь(255, 127, 80, .5). Изучите цвет элемента. Обратите внимание, как отображаются стили для данного элемента в инструментах веб-разработчика.

6. Параграфу task-6 присвойте размер шрифта 18рх, семейство шрифта Arial, sans-serif.

Услугами бельгийской баржи класса люкс (на илл.) пользовались цари, короли и императоры. Гентская баржа

7. Параграфу task-7 присвойте размер шрифта 18рх, цвет шрифта #333, семейство Georgia,serif жирность - 800.

Территориальная еди ща Индии имеет в названии три союза «и».

8. Согласитесь, каждый раз писать цвет, жирность шрифта - утомительно получившиеся классы узко специализированы. Ведь, если мне нужно сделать такой же текст, но не жирный - то придется делать практически дубликат класс. Глупо, и нарушает принцип DRY. Давайте сделаем класс bold, в котором будет всего одна строка font-weight: bold, и класс .сога который содержит свойство background: color.

9. Испытаем созданные классы. Допишите к task-9 дополнительно классы bold, coral. A самому task-9 задайте font-size 16px.

Физик-теоретик считает, что математическая формула с 14 членами в одном выражении - самое красивое, что она сделала в жизни.

10. Добавьте классу task-10 следующие свойства - font-style italic, text-decoration undeline, text-transform uppercase.
